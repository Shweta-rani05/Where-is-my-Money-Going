# Where Is My Money Going?

An AI-powered Personal Finance & Expense Tracking application built using the MERN stack (MongoDB, Express, React, Node.js) with TypeScript.

## Features

- **JWT Authentication**: Secure user register, login, logout, and protected client/server routes.
- **Transaction CRUD**: Add, edit, delete, search, and filter personal financial transactions.
- **Budget Tracking**: Establish monthly limits per category and receive alerts.
- **Savings Goals**: Set goals and track visual progress.
- **Interactive Visualizations**: Beautiful dashboards and charts showing categories, income vs. expenses, and monthly trends.
- **AI-Powered Insights**: Get financial analysis, budget advice, and chat with an AI model regarding your spending.
- **Reports**: Generate and export reports as CSV/PDF.

## Tech Stack

- **Frontend**: React (Vite), TypeScript, Tailwind CSS, TanStack Query, Framer Motion, Recharts, React Hook Form, Zod.
- **Backend**: Node.js, Express.js, TypeScript, Mongoose, JWT, bcryptjs, Helmet, Morgan, Cors, Express Validator.
- **AI**: Gemini API / OpenAI API.
